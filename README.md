# IA-Neurosimbólica en Grandes Modelos de Lenguaje

Presentación para seminario AIAA.

## Estructura de trabajo simultáneo

### 1) Objetivo común
Construir una presentación clara sobre IA neurosimbólica aplicada a LLMs, equilibrando base teórica, estado del arte y casos prácticos.

### 2) División en frentes paralelos
- **Frente A — Fundamentos**: definición de IA neurosimbólica, motivación y conceptos clave.
- **Frente B — Arquitecturas y métodos**: integración simbólico-neuronal, recuperación de conocimiento, razonamiento y planificación.
- **Frente C — Evidencia y evaluación**: benchmarks, métricas, limitaciones y comparación con enfoques puramente neuronales.
- **Frente D — Aplicaciones y demo**: casos de uso en LLMs, riesgos, consideraciones éticas y cierre.

### 3) Responsabilidades por frente
Cada frente debe entregar:
1. 3–5 diapositivas.
2. Mensajes clave (máximo 3 por diapositiva).
3. Referencias bibliográficas verificables.
4. Guion breve de exposición (2–3 minutos por persona).

### 4) Flujo de integración
1. Trabajo en paralelo por frente.
2. Revisión cruzada entre frentes (coherencia y no solapamiento).
3. Integración en un único orden narrativo:
   - Problema → Enfoque neurosimbólico → Evidencia → Aplicaciones.
4. Ensayo general con control de tiempos y ajustes finales.

### 5) Criterios de calidad
- Consistencia visual y terminológica.
- Rigor técnico sin perder claridad.
- Fuentes actuales y trazables.
- Conclusiones accionables para investigación futura.

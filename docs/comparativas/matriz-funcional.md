# Matriz funcional

!!! tip "TL;DR"
    Además de la taxonomía de Kautz, conviene clasificar sistemas por el rol del
    componente neuronal: formalizador, reparador o heurística.

## Matriz

| | Búsqueda determinista sobre representación formal | Búsqueda guiada por heurística |
|---|---|---|
| LLM como formalizador | LLM+P, DUPLEX, Logic-LM | CEGIS |
| Modelo neuronal como heurística | — | AlphaGeometry2, NELLIE |

## Cómo usarla

Esta matriz no sustituye a Kautz. Sirve para entender quién hace qué dentro del
pipeline.

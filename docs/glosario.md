# Glosario

!!! tip "TL;DR"
    Términos mínimos para navegar la wiki sin perderse.

| Término | Definición |
|---|---|
| NeSy | IA neurosimbólica: integración de modelos neuronales y razonamiento simbólico. |
| LLM | Gran Modelo de Lenguaje, normalmente transformer autoregresivo. |
| PDDL | Lenguaje formal para problemas de planificación. |
| SMT | Satisfiability Modulo Theories, satisfacibilidad bajo teorías matemáticas. |
| Z3 | Solver SMT ampliamente usado. |
| FOL | Lógica de Primer Orden. |
| DDAR | Motor deductivo usado en AlphaGeometry. |
| Self-refinement | Bucle de reparación con feedback del solver. |
| CEGIS | Síntesis inductiva guiada por contraejemplos. |
| Soundness | Propiedad de no derivar conclusiones inválidas desde premisas válidas. |
| Grounding | Anclaje semántico de símbolos a mundo, reglas o dominio. |
| Explainability laundering | Uso de explicaciones formales para legitimar premisas sesgadas. |
